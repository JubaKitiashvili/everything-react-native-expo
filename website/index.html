How it works

Up and running in 60 seconds

1

Run the installer

One command detects your project type — Expo managed, bare RN, or monorepo — and sets everything up.

2

Choose your hook profile

minimal for vibe coding, standard for balanced quality, or strict for production enforcement.

3

Pick MCP integrations

Connect simulator control, GitHub, Supabase, Firebase, Figma, Sentry — whatever your stack needs.

4

Start building

Use /plan to architect features, /tdd for test-driven development, or /code-review for quality audits.

Hook profiles

Quality enforcement your way

Three profiles that control how aggressively hooks check your code. Switch anytime.

⚡

minimal

Fast iteration with no friction. Perfect for prototyping and vibe coding sessions.

  • Basic lint checks
  • No blocking hooks
  • Maximum speed
Recommended
⚖

  • Platform-specific checks
  • Security scanning
  • Bundle size monitoring
